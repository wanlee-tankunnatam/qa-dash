# Workflow Flags Board

**วิธีใช้:**
- แต่ละ role เพิ่ม flag ด้วยรูปแบบ `- [ ] [FROM→TO] <งาน>`
- role ที่รับงานเปลี่ยน `[ ]` เป็น `[x]` เมื่อทำเสร็จ
- ลบ flags ที่ `[x]` ออกเมื่อ session ใหม่เริ่มต้น
